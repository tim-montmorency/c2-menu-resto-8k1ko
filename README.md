## Description
 
[Ce projet présente le menu d’un restaurant fictif nommé *La Table Urbaine*.  
Le menu est divisé en trois catégories : Entrées, Plats principaux et Desserts.  
L’objectif était de créer une structure HTML sémantique claire et un design simple.]
 
## Choix techniques
 
**Palette de couleurs :**
 
- Primaire : #2c3e50
- Secondaire : #e74c3c
- Texte : #333  
- Fond : #f9f9f9  
**Police utilisée :**
 
- [ Arial ]
 
**Décisions CSS notables :**
 
- Utilisation de Flexbox pour aligner le nom des plats et les prix
- Badges stylisés (POPULAIRE, VÉGÉ, NOUVEAU)
- Effet hover sur les items du menu pour améliorer l’interaction
- Structure claire avec des sections distinctes
